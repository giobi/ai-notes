# AI Notes - Informazioni sul Progetto

## Panoramica

**AI Notes** è un progetto didattico per un corso di introduzione all'Intelligenza Artificiale destinato all'Università della Terza Età.

## Obiettivi

Creare materiale didattico accessibile e comprensibile che copra:
1. Introduzione all'IA: dal sogno alla realtà
2. Cos'è l'Intelligenza Artificiale: spiegazione semplice e accessibile
3. Applicazioni Pratiche: l'IA nella vita quotidiana
4. Aspetti Etici e Futuro: un dibattito aperto

## Durata del Corso

Il corso è strutturato per una durata totale di **90 minuti**, comprese le domande e le discussioni.

## Target di Pubblico

- Studenti dell'Università della Terza Età
- Persone interessate a comprendere l'IA senza background tecnico
- Chiunque voglia un'introduzione accessibile all'Intelligenza Artificiale

## Tecnologie Utilizzate

- **reveal.js**: Per presentazioni web interattive
- **GitHub Pages**: Hosting delle presentazioni
- **Jekyll**: Generazione del sito statico
- **Markdown**: Formato principale per i contenuti

## Struttura delle Presentazioni

Le presentazioni sono disponibili in formato:
- **Web interattivo** (reveal.js) - accessibile via browser
- **PDF/PowerPoint** (nella cartella `/slides`)

### Presentazioni Disponibili

1. **Congresso Verbania** - Presentazione completa con tutti i 4 capitoli
2. **Introduzione** - Dal sogno alla realtà
3. **Cos'è l'IA?** - Definizioni ed esempi

## Flusso di Lavoro

```
Idee → Appunti → Capitoli → Slides/Presentazioni
  ↓
Risorse (per approfondimenti)
```

## Sviluppo e Manutenzione

### Come Contribuire

1. Aggiungi nuove idee in `/appunti`
2. Organizza contenuto maturo in `/capitoli`
3. Raccogli risorse in `/risorse`
4. Crea presentazioni in `/slides` o `/presentation`
5. Documenta modifiche in `/docs/diary`

### Convenzioni

- Usa nomi file descrittivi in formato kebab-case
- Documenta le modifiche significative
- Mantieni la struttura organizzata
- Pensa sempre al pubblico non tecnico

## Note Tecniche

### GitHub Pages Configuration

Il sito è configurato con Jekyll (_config.yml):
- Base URL: https://giobi.github.io/ai-notes
- Lingua: Italiano
- Plugin: jekyll-feed, jekyll-sitemap
- Le presentazioni sono incluse automaticamente

### reveal.js

Le presentazioni interattive usano reveal.js con:
- Navigazione con frecce direzionali
- `ESC` per overview
- `F` per fullscreen
- `S` per note del relatore
